Android Sign in to Wi-Fi Network Settings

Sign in to Wi-Fi Network is a system window in Android. Some users are currently experiencing problems when they connect via any Captive Portal Solution and the Sign in to Wi-Fi Network window is automatically closed. Which means that no advertisers web page is displayed and/or no custom success page is displayed.

The solution uses an experimental workaround to improve the user experience for the users using Android devices. More information.

Note: Please note that the Android OS is heavily fragmented and many OEM vendors use a different name for the Sign in to Wi-Fi Network.

Enable this workaround for this brand.

When Media Network is enabled, you cannot disable this workaround. Disable Media Network to switch this workaround off.
